## About

Phoenix Bot (PHX1) was my first attempt at building an automated Solana memecoin discovery system.

The goal was simple:

- Discover newly launched tokens
- Track market structure in real time
- Identify potential retracement opportunities
- Display everything in a live dashboard

This repository represents my first large AI-assisted software project while learning modern LLM-driven development.

Although later versions of Phoenix significantly improved the architecture, PHX1 remains public as a snapshot of where the project began.

---

## Features

- 📡 Live token discovery
- 📈 Real-time market tracking
- 📊 Interactive dashboard
- 🔎 Token database
- 📉 ATH / retracement monitoring
- 🔄 Live websocket updates
- 🗂 Historical token tracking
- ⚡ AI-assisted architecture

## Why Open Source?

I intentionally left this version public because I believe showing the evolution of a project is more valuable than pretending every first version was perfect.

This repository documents my learning process while building increasingly complex event-driven systems.

## Lessons Learned

Building PHX1 taught me:

- Event-driven architecture
- WebSocket pipelines
- State management
- Dashboard development
- API integrations

- AI-assisted software engineering

## Current Status

PHX1 is no longer actively developed.

Development has moved to newer private iterations with redesigned architecture, improved data pipelines, stronger filtering, and substantially higher reliability.

This repository is preserved for educational purposes and as part of my engineering portfolio.
